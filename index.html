Vivo Rdfify URL Bookmarklet

This bookmarklet transforms the url of a vivo profile to the url that displays rdf. This is useful for learning about what the data in Vivo looks like as well debugging information.

For example, running the bookmarklet while at the page: http://vivo.ufl.edu/individual/UniversityofFlorida will change the page to http://vivo.ufl.edu/individual/UniversityofFlorida/UniversityofFlorida.rdf

To install this bookmarklet drag the following link onto your toolbar: RdfifyUrl
